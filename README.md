Project Overview: 
Detailed explanation of ThermalML's physics-informed Random Forest + XGBoost
ensemble model for electronic heat sink sizing and fan recommendation.
Key Features:
Covers the prediction engine, Ansys Icepak CFD verification
integration, benchmark metrics, 11-slide academic presentation module, session
logging with CSV export, and academic credits.
Model Architecture & Data Pipeline:
Formally details all 8 input features (), 8 output target variables (),
data preprocessing protocols, and ensemble soft-voting equations.
Performance Benchmarks: 
Lists metrics (
 for CFM, 
 for Fin Count) and
 classification accuracies (for Material & Surface Finish).
Technology Stack: 
React 19, TypeScript, Vite, Tailwind CSS v4, 
Motion, Lucide Icons, and Recharts.
Local Setup & GitHub Commands:
Complete step-by-step terminal instructions for cloning, installing dependencies,
starting local dev, building for production, and pushing code to GitHub.
